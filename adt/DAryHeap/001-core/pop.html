Pop the root from a d-ary heap

Hypothesis : i < j

Parameters:

  • arity must be an int.
    (arity of the heap)

  • compare must be a function.
    (the comparison function)

  • swap must be a function.
    (the swap function)

  • a must be an array.
    (the array where the heap is stored)

  • i must be an int.
    (is the root)

  • j must be an int.
    (- 1 is the last leaf)

decrement size of heap

--j;

put last leaf at root

popped = a[i]; a[i] = a[j];

percolate down the new root

daryheap.percolatedown( arity, compare, swap, a, i, j, i );

return popped; };
